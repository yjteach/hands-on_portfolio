### ヘッダー部分CSS作成実習（5−10分）
要件
* 見本と同じレイアウトになるようにCSSを書いてみる。
* 余裕のある人は、色を変えてみたりアイコンを好きなものに変えて。

ヒント
* 左右の振り分け：float
* 文字サイズ：font-size
* 画像と文字の高さ調整：vertical-align
* 余白調整：padding、margin
