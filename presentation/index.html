よく使うタグ

タグ名 説明
見出しタグ h1からh6。数字が小さい方が大見出し。
pタグ 段落ごとにpタグで囲む。
aタグ 他ページなどにリンクさせる時に使う。
imgタグ 画像を使いたい時に使う
ul,liタグ リストを表示したい時に使う
dl,dt,ddタグ 定義リストを表示したい時に使う
divタグ htmlとしての意味は無く、CSSでの装飾用に囲みたい時に使う
詳しくはWEBで！

演習2（CSS）

ヘッダー部分のCSS作成(10分)

css/style.css


/* header -------------------------- */
ここに書く
要件
  • 見本と同じレイアウトになるようにCSSを書く
  • 余裕のある人は、色を変えてみたりアイコンを
    好きなものに変えてみる。

ヒント
左右の振り分け float
文字サイズ font-size
画像と文字の高さ調整 vertical-align
余白調整 padding、margin
マウスオン時のリンク色変更 a:hover
リンクの下線 text-decoration

解答例


/* header -------------------------- */
#top {
  padding: 15px 0;
}

#top h1 {
  font-size: 140%;
  font-weight: bold;
}

#top h1 img {
  vertical-align: middle;
  margin-right: 10px;
}

#top ul {
  float: right;
  margin-top: -32px;
}

#top ul li {
  float: left;
  padding-left: 20px;
}

#top ul li a {
  display: block;
  text-decoration: none;
  color: #000000;
  font-size: 130%;
}

#top ul li a:hover {
  color: #cc0000;
  text-decoration: underline;
}
